Mystery Word Game

A small word-guessing experiment I built while learning to code.
Choose a mode and try to guess the hidden word.
Each mode hides a different type of word.
How it works:
_ ‚Üí letter is not in the word
lowercase ‚Üí letter is in the word, but in the wrong position
UPPERCASE ‚Üí letter is in the correct position
